
<p>You'll need to create a <a href="https://github.com/settings/applications/new">github application</a>.</p>

<p>Set the auth callback URL to <code>{$this->getConfigUrl()}&oauth=1</code>.</p>
<p>Once that's done, fill out this form:</p>


<form action="{$this->getConfigUrl()}&setconfig=1" method="post">
<p>Client ID: <input type="text" name="client_id" value="{$this->cfg.client_id|escape}" /></p>
<p>Client Secret: <input type="text" name="client_secret" value="{$this->cfg.client_secret|escape}" /></p>
<p><input type="submit" value="Save Settings" /></p>
</form>
